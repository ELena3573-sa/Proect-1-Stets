# Proect-1 Разработка приложения Stets для умного дома.
Stets — это учебный проект, разработанный в рамках курса "Системный аналитик".
Приложение разработано чтобы централизованно управлять устройствами умного дома, комнатами, домами.

Как системный аналитик, я проанализировала интервью с заказчиком и пользователями и выделила требования.
Разработала в Miro карту пользовательских историй и MVP, написала критерии и
сценарии приёмки с помощью языка Gherkin, в Draw.io разработала DFD контекстного и
логического уровня, проектировала ER-диаграмму, привела  её к 3НФ, написала словарь
данных, нарисовала диаграмму структуры интерфейсов, разработала прототипы в Figma и
написала ПМИ.
